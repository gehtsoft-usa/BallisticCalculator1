Understanding the output

Brief

What a trajectory point contains and how to read it.

Details

TrajectoryCalculator.Calculate returns an array of TrajectoryPoint, one per output step. Every physical value is a Measurement<T>, so you choose the unit when you read it (for example point.Velocity.In(VelocityUnit.FeetPerSecond)).

Property

Meaning

Time

Time of flight to the point.

Distance

Distance along the line of sight (the sight axis).

DistanceFlat

Horizontal distance from the muzzle.

Velocity

Projectile speed.

Mach

Speed relative to the local speed of sound.

Drop

Vertical position relative to the line of sight. At the muzzle it equals minus the sight height.

DropFlat

Vertical position relative to the muzzle.

Windage

Horizontal deflection. Left is positive, right is negative. Includes spin drift when it is modelled.

Energy

Kinetic energy.

DropAdjustment

Angular correction for the drop at this distance (scope come-up).

WindageAdjustment

Angular correction for the windage at this distance.

OptimalGameWeight

Litz optimal game weight estimate.

A crossed reference: Drop is measured from the sighting line, so it is negative below the line of sight and around zero at the zero range; DropFlat is measured from the bore line and is almost always negative downrange.

var ammo = new Ammunition(
    weight: new Measurement<WeightUnit>(168, WeightUnit.Grain),
    ballisticCoefficient: new BallisticCoefficient(0.223, DragTableId.G7),
    muzzleVelocity: new Measurement<VelocityUnit>(2700, VelocityUnit.FeetPerSecond));
 
var rifle = new Rifle(
    sight: new Sight(new Measurement<DistanceUnit>(1.5, DistanceUnit.Inch),
                     Measurement<AngularUnit>.ZERO, Measurement<AngularUnit>.ZERO),
    zero: new ZeroingParameters(new Measurement<DistanceUnit>(100, DistanceUnit.Yard), null, null));
 
var atmosphere = new Atmosphere();
var calc = new TrajectoryCalculator();
 
var shot = new ShotParameters
{
    MaximumDistance = new Measurement<DistanceUnit>(600, DistanceUnit.Yard),
    Step = new Measurement<DistanceUnit>(200, DistanceUnit.Yard),
    SightAngle = calc.SightAngle(ammo, rifle, atmosphere),
};
 
foreach (var p in calc.Calculate(ammo, rifle, atmosphere, shot))
    Console.WriteLine($"{p.Time.TotalSeconds:0.000} {p.Distance.In(DistanceUnit.Yard),4:N0} " +
        $"{p.Velocity.In(VelocityUnit.FeetPerSecond),5:N0} {p.Mach,4:N2} " +
        $"{p.Drop.In(DistanceUnit.Inch),7:N1} {p.Energy.In(EnergyUnit.FootPound),5:N0} " +
        $"{p.DropAdjustment.In(AngularUnit.MOA),6:N2}");
