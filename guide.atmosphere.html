Atmosphere, wind, and drift

Brief

Non-standard air, multiple wind zones, and spin drift.

Details

Three additions turn the bare trajectory into a realistic one.

Atmosphere

Build an Atmosphere from altitude, pressure, temperature, and humidity. Two things to remember: **humidity is a fraction from 0 to 1** (not a percentage), and the four-argument constructor treats the pressure as the station pressure at that altitude (use the overload with pressureAtSeaLevel: true to pass a sea-level-corrected value).

Wind

Wind is given as a velocity and a direction (0 deg blows toward the target, 90 deg blows from the right). Pass an array to model several zones; each applies out to its MaximumRange, and the last (or one with no range) applies to the end. The zones must be ordered by increasing range.

Spin drift

Spin drift is modelled only when the Rifle has Rifling and the ammunition has both a bullet diameter and a bullet length. It is folded into Windage (there is no separate output column).

var ammo = new Ammunition(
    weight: new Measurement<WeightUnit>(168, WeightUnit.Grain),
    ballisticCoefficient: new BallisticCoefficient(0.223, DragTableId.G7),
    muzzleVelocity: new Measurement<VelocityUnit>(2700, VelocityUnit.FeetPerSecond),
    bulletDiameter: new Measurement<DistanceUnit>(0.308, DistanceUnit.Inch),
    bulletLength: new Measurement<DistanceUnit>(1.2, DistanceUnit.Inch));
 
var rifle = new Rifle(
    sight: new Sight(new Measurement<DistanceUnit>(2.0, DistanceUnit.Inch),
                     Measurement<AngularUnit>.ZERO, Measurement<AngularUnit>.ZERO),
    zero: new ZeroingParameters(new Measurement<DistanceUnit>(100, DistanceUnit.Yard), null, null),
    rifling: new Rifling(new Measurement<DistanceUnit>(11.25, DistanceUnit.Inch), TwistDirection.Right));
 
var atmosphere = new Atmosphere(
    altitude: new Measurement<DistanceUnit>(5000, DistanceUnit.Foot),
    pressure: new Measurement<PressureUnit>(24.9, PressureUnit.InchesOfMercury),
    temperature: new Measurement<TemperatureUnit>(40, TemperatureUnit.Fahrenheit),
    humidity: 0.30);
 
var wind = new[]
{
    new Wind(new Measurement<VelocityUnit>(10, VelocityUnit.MilesPerHour),
             new Measurement<AngularUnit>(90, AngularUnit.Degree),
             new Measurement<DistanceUnit>(500, DistanceUnit.Yard)),
    new Wind(new Measurement<VelocityUnit>(5, VelocityUnit.MilesPerHour),
             new Measurement<AngularUnit>(45, AngularUnit.Degree)),
};
 
var calc = new TrajectoryCalculator();
var shot = new ShotParameters
{
    MaximumDistance = new Measurement<DistanceUnit>(1000, DistanceUnit.Yard),
    Step = new Measurement<DistanceUnit>(100, DistanceUnit.Yard),
    SightAngle = calc.SightAngle(ammo, rifle, atmosphere),
};
 
foreach (var p in calc.Calculate(ammo, rifle, atmosphere, shot, wind))
    Console.WriteLine($"{p.Distance.In(DistanceUnit.Yard),5:N0} " +
        $"{p.Velocity.In(VelocityUnit.FeetPerSecond),6:N0} " +
        $"{p.Drop.In(DistanceUnit.Inch),8:N1} " +
        $"{p.Windage.In(DistanceUnit.Inch),7:N1}");
