class DrgDragTableFactory

Parents

object

Brief

Synthesizes a custom drag table from a standard base drag curve and an effective ballistic-coefficient-vs-Mach profile.

Details

The synthesized curve is Cd_custom(M) = Cd_base(M) / BC(M), where BC(M) is interpolated from the supplied knots.

The result is a custom GC table; run it with a BallisticCoefficient value of 1.0 and table GC so the engine applies exactly the intended per-Mach drag.
