Custom drag: GC and .drg tables

Brief

Use a measured or custom drag curve instead of a standard G-table.

Details

When a standard G1/G7 curve does not describe a bullet well, supply your own drag curve. A custom curve uses the GC table id and is passed directly to both SightAngle and Calculate.

A custom drag table in code

Derive from DragTable, return DragTableId.GC, and feed it drag points (Mach, drag coefficient). The ammunition uses a BallisticCoefficient with the GC table id.

class MyDrag : DragTable
{
    public override DragTableId TableId => DragTableId.GC;
    private static readonly DragTableDataPoint[] Points =
    {
        new DragTableDataPoint(0.00, 0.180), new DragTableDataPoint(0.60, 0.129),
        new DragTableDataPoint(0.80, 0.136), new DragTableDataPoint(0.90, 0.177),
        new DragTableDataPoint(1.00, 0.427), new DragTableDataPoint(1.05, 0.452),
        new DragTableDataPoint(1.20, 0.429), new DragTableDataPoint(1.50, 0.394),
        new DragTableDataPoint(2.00, 0.339), new DragTableDataPoint(2.50, 0.297),
        new DragTableDataPoint(3.00, 0.250), new DragTableDataPoint(5.00, 0.180),
    };
    public MyDrag() : base(Points) { }
}
 
var table = new MyDrag();
var ammo = new Ammunition(
    weight: new Measurement<WeightUnit>(168, WeightUnit.Grain),
    ballisticCoefficient: new BallisticCoefficient(0.5, DragTableId.GC),
    muzzleVelocity: new Measurement<VelocityUnit>(2700, VelocityUnit.FeetPerSecond));
 
var rifle = new Rifle(
    sight: new Sight(new Measurement<DistanceUnit>(1.5, DistanceUnit.Inch),
                     Measurement<AngularUnit>.ZERO, Measurement<AngularUnit>.ZERO),
    zero: new ZeroingParameters(new Measurement<DistanceUnit>(100, DistanceUnit.Yard), null, null));
 
var atmosphere = new Atmosphere();
var calc = new TrajectoryCalculator();
var shot = new ShotParameters
{
    MaximumDistance = new Measurement<DistanceUnit>(1000, DistanceUnit.Yard),
    Step = new Measurement<DistanceUnit>(100, DistanceUnit.Yard),
    SightAngle = calc.SightAngle(ammo, rifle, atmosphere, table),
};
 
foreach (var p in calc.Calculate(ammo, rifle, atmosphere, shot, null, table))
    Console.WriteLine($"{p.Distance.In(DistanceUnit.Yard),5:N0} " +
        $"{p.Velocity.In(VelocityUnit.FeetPerSecond),6:N0} " +
        $"{p.Drop.In(DistanceUnit.Inch),8:N1}");

Loading a radar .drg file

A measured Doppler-radar curve is usually distributed as a .drg file. Load it with DrgDragTable.Open and use it exactly like the custom table above (pass it to SightAngle and Calculate):

Loading a .drg file Show

Loading a .drg file Hide

var table = DrgDragTable.Open("308-168gr.drg");
// then use `table` the same way as a custom DragTable
