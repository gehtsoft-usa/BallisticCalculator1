static public method DrgDragTableFactory.Build

Brief

Builds a custom drag table from a base curve and a Mach-to-BC profile.

Declaration
C#
DrgDragTable  Build (AmmunitionLibraryEntry ammunition, DragTableId baseTable, IEnumerable<BcAtMach> bcCurve)

Parameters
ammunition

Ammunition metadata (name, weight, diameter) attached to the resulting table.
baseTable

The standard drag curve to scale (for example G1 or G7). Must not be GC.
bcCurve

The Mach-to-effective-BC knots. Order does not matter and at least one knot is required. BC is interpolated linearly between knots and held flat beyond the end knots.
